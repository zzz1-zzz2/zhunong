# 农业数据监测图片

这里应该放置从豆包AI生成的农业数据监测图片：

## 图片列表

### 1. 数据分析表图片
- **文件名**: `data-analysis-chart.png`
- **描述**: 大型白色数据表，包含彩色柱状图（绿色、黄色、橙色、红色）和数据网格
- **用途**: 用于互动剧中的数据分析场景

### 2. 果园工作场景
- **文件名**: `orchard-work-scene.png`
- **描述**: 工作人员穿着红色背心，带有圆形白色徽章，果园中的水泵组件
- **用途**: 用于展示果园日常工作场景

### 3. 土壤监测设备
- **文件名**: `soil-sensor-device.png`
- **描述**: 安装在滴灌管线上的土壤传感器，带数字显示屏幕
- **用途**: 用于展示智能农业监测设备

### 4. 实验样本标记
- **文件名**: `experiment-samples.png`
- **描述**: 两个透明袋，分别标记为"55*"和"26*"，田间试验场景
- **用途**: 用于展示农业实验数据收集

## 使用说明

1. 下载豆包AI生成的图片
2. 重命名为上述对应的文件名
3. 放置在此目录中
4. 在代码中引用本地路径

## 引用方式

在React组件中引用：
```jsx
<img src="/images/agricultural-data/data-analysis-chart.png" alt="数据分析" />
```
